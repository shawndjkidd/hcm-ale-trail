Visit 9 craft breweries in Ho Chi Minh City, collect stamps, and earn a FREE hat!

## Features

- **9 Breweries**: Explore the best craft beer spots in HCMC
- **Digital Stamps**: Collect stamps by entering secret codes at each brewery
- **Progress Tracking**: Track your journey with a visual progress bar
- **Multi-language**: Support for English, Vietnamese, Korean, and Japanese
- **QR Code Ready**: Scan QR codes at breweries or ask bartenders to enter codes
- **Mobile-Friendly**: Fully responsive design for on-the-go beer adventures

## Breweries on the Trail

1. **BiaCraft** - District 3
2. **Heart of Darkness** - District 1
3. **Deme Brewing** - District 3
4. **Steersman Brewery** - Tan Binh
5. **East West Brewing** - District 1
6. **Pasteur St Brewing** - District 1
7. **Rooster Beers** - District 1
8. **7 Bridges Brewing** - District 1
9. **Belgo Saigon** - District 1

## How to Use

1. Visit any brewery on the Ale Trail
2. Order a beer
3. Ask your bartender to:
   - Show you the QR code to scan, OR
   - Enter the secret code for you
4. Collect your stamp!
5. Collect all 9 stamps to earn your FREE hat!

**Take your time** - finish the trail in a day, a week, or a year!

## Technology Stack

- React 18
- Tailwind CSS
- Lucide React Icons
- Deployed on Vercel

## Local Development

```bash
# Clone the repository
git clone https://github.com/yourusername/hcm-ale-trail.git

# Navigate to directory
cd hcm-ale-trail

# Open index.html in your browser
# Or use a local server like:
python -m http.server 8000
```

## Deployment

This app is deployed on Vercel. Any push to the main branch automatically triggers a deployment.
